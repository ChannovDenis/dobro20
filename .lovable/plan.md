
# Переработка интерфейса Добросервис

## Обзор изменений

Три ключевых экрана будут полностью переработаны по референсу wibes-dobro.lovable.app:

---

## 1. Лента в стиле TikTok

**Текущее состояние:** Классическая лента с карточками в столбик, заголовками секций, сеткой сервисов.

**Новый дизайн:**
- Полноэкранные вертикальные карточки (100vh)
- Вертикальный скролл с snap-эффектом (как в TikTok)
- Фоновое изображение/видео занимает весь экран
- Размытый фон изображения для эффекта глубины
- Информация overlay внизу карточки (название, автор, теги)
- Правая панель с иконками (лайк, комментарии, поделиться)
- Индикатор "свайп" для навигации
- Каждая 7-я карточка -- промо сервиса или мини-приложения

**Технические детали:**
```
- CSS scroll-snap-type: y mandatory
- Высота карточки: 100dvh (динамическая высота viewport)
- Overlay с градиентом снизу
- Плавающий TopBar с прозрачным фоном
- Скрытие BottomNav при скролле (опционально)
```

**Структура данных для ленты:**
- Расширенные contentItems с полноэкранными изображениями
- Поле type: "content" | "service-promo" | "miniapp-promo"
- Теги для категоризации
- Данные для промо-карточек сервисов

## 2. Чат -- минималистичный дизайн

**Текущее состояние:** Сетка из 6 быстрых действий 3x3, большой welcome-блок с иконкой.

**Новый дизайн:**
- Чистое пустое пространство при старте
- Компактный заголовок "Добросервис AI"
- Минимальные подсказки: 3-4 короткие фразы-чипа горизонтально
  - "Спросить юриста" 
  - "Про здоровье"
  - "Проверить смету"
- Подсказки исчезают после первого сообщения
- Большое поле ввода с placeholder "Напишите что угодно..."
- Убрать кнопку "+" шаблонов (или сделать менее заметной)

**Технические детали:**
- QuickActions переделать в компактные чипы
- Минимум визуального шума
- Фокус на input

---

## 3. Настройки как супер-апп

**Текущее состояние:** Простой список секций с переключателями.

**Новый дизайн:**

### Шапка профиля (Hero)
- Большой аватар по центру с glow-эффектом
- Имя и статус подписки
- Кнопка редактирования профиля

### Сетка сервисов (SuperApp Grid)
- Иконочная сетка 4x3 всех доступных сервисов/фич
- Каждый элемент: иконка + название
- Цветовые акценты по категориям
- При клике -- переход к деталям сервиса

### Секции меню (Collapsible)
- "Мои подписки" -- Премиум статус, продление
- "Уведомления" -- Push, Email, Промо
- "Безопасность" -- Пароль, устройства, 2FA
- "Приложение" -- Тема, язык, кэш
- "О приложении" -- Версия, ссылки

### Подвал
- Кнопка выхода
- Ссылки на соцсети
